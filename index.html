A beautiful widget for your site

Better GitHub Widget is the best looking widget to display your GitHub projects, it simply blends with your theme.

Client side magic

All of the magic of this widget happens on the user's browser, so it does not stress your server.

Installation

  1. Go to the wordpress.org page
  2. Upload the contents of the zip (including the folder) to the /wp-content/plugins/ directory
  3. Activate the plugin through the 'Plugins' menu in WordPress
  4. Place the widget to a widget area and set your GitHub username
